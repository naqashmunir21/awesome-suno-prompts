## Production Tips

### 🎛️ **Key Production Elements by Subgenre**

| Subgenre | Signature Sound | BPM Range | Energy Curve |
|----------|-----------------|-----------|--------------|
| **Big Room** | Supersaws + kick | 126-130 | Build → Drop → Build |
| **Progressive House** | Arpeggios | 120-128 | Gradual journey |
| **Dubstep** | Wobble bass | 140 (70 feel) | Buildup → Drop chaos |
| **Trance** | Soaring leads | 130-145 | Emotional breakdown → Release |
| **Techno** | Hypnotic groove | 120-140 | Continuous energy |
| **Future Bass** | Vocal chops + saws | 140-160 | Emotional + energetic |

### 🎵 **Mix/Master Considerations**

- **Festival Bangers:** Brick-wall limiting, maximum loudness, mono-compatible bass
- **Progressive House:** Dynamic range preserved, stereo width, clear mix
- **Dubstep:** Sub-bass dominance, mid-range aggression, headroom for drops
- **Trance:** Bright top-end, driving kick/bass relationship, emotional space
- **Techno:** Tight kick/bass, minimal frequency clutter, hypnotic repetition
- **Future Bass:** Colorful stereo field, bright supersaws, vocal clarity

### 🥁 **Drum Programming**

- **Big Room:** Punchy kick on every beat, minimal hi-hats, crash on drops
- **Progressive House:** Four-on-floor kick, tight hi-hats, percussion layers
- **Dubstep:** Half-time feel, snare on 3, sub-bass on 1 and 3
- **Trance:** Driving kick, rolling hi-hats, build-up fills
- **Techno:** Minimal variation, hypnotic repetition, industrial percussion
- **Future Bass:** Syncopated rhythms, trap-influenced hi-hats, energetic fills

### 📊 **Drop Structure Templates**

**Big Room Drop:**
1. 8-bar build (risers + tension)
2. 1-bar silence/filter
3. Drop: Kick + bass + supersaw (16 bars)
4. Breakdown (8 bars)
5. Second drop variation (16 bars)

**Progressive House Journey:**
1. Intro groove (16-32 bars)
2. Add elements gradually (32 bars)
3. First drop (16 bars)
4. Breakdown (16-32 bars)
5. Peak drop (16-32 bars)
6. Outro (16 bars)

**Dubstep Drop:**
1. Build vocal + drums (8 bars)
2. Final riser (4 bars)
3. Drop: Wobble bass chaos (8 bars)
4. Second drop variation (8 bars)
