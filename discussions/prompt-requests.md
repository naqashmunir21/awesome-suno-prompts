# Prompt Requests

Use this thread to request tailored prompts. Please include the following to help contributors:

- **Genre** (e.g., Pop, Hip-Hop, EDM)
- **Mood / Goal** (e.g., uplifting chorus, moody bridge, radio-ready verse)
- **Tempo / BPM** (optional)
- **Reference artists or tracks** (optional)
- **Any production cues** (instruments, arrangement, vocal style)

Example request:

```
Genre: Pop
Mood: Bright, euphoric chorus
Tempo: 100 BPM
Reference: Dua Lipa, modern disco
Production cues: tight synth bass, four-on-the-floor drums, lush vocal harmonies
```

Contributors: please reply with one or more prompt variations and mark which one worked best for you.
