# Welcome to Awesome Suno Prompts

Welcome — thank you for checking out **Awesome Suno Prompts**. This repository is a community-curated collection of 1000+ professional Suno AI style prompts organized by genre, mood, and production tips.

How to use this space:

- Ask questions about using the prompts.
- Request genre- or mood-specific prompts in **#prompt-requests** (use the Prompt Requests discussion).
- Share results (audio, screenshots, lyrics) in **Show & Tell**.
- Open a PR with new prompts or improvements — see `CONTRIBUTING.md` for formatting.
