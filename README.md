## 💡 Best Practices

### ✅ DO:
- **Be specific with instrumentation** - Instead of "guitar", say "crunchy distorted guitar" or "clean fingerpicked acoustic"
- **Include BPM and Key** - Helps Suno understand tempo and melodic framework
- **Use production terminology** - Reverb, saturation, compression, sidechain, etc.
- **Specify vocal style** - Raspy, smooth, breathy, powerful, melismatic, etc.
- **Mention mixing approach** - Radio-ready, lo-fi, live recording, analog warmth, etc.
- **Use arrow notation for evolution** - "soft verse → explosive chorus"
- **Stay under 950 characters** - Suno truncates longer prompts

### ❌ DON'T:
- Use vague terms like "good song" or "nice beat"
- Mix contradictory styles (e.g., "lo-fi + stadium production")
- Forget to specify Suno version (V4.5 vs V5 have different strengths)
- Ignore genre conventions (every genre has expected elements)
- Overload with too many ideas (focus on 3-5 core elements)

## 🚫 Common Mistakes

### ❌ Problem: Generic Prompt
```
Make a pop song with good vocals and catchy melody
```
**Why it fails:** Too vague, no production details, no specific sound

### ✅ Solution: Specific Prompt
```
Infectious synth-pop, female powerhouse vocals with layered harmonies,
pulsing 808 bass, euphoric build-ups, modern radio production,
BPM: 128, Key: C Major
```
**Why it works:** Clear genre, specific vocals, production details, technical specs

---

### ❌ Problem: Contradictory Elements
```
Lo-fi bedroom aesthetic with stadium reverb and arena production
```
**Why it fails:** Lo-fi and stadium production are opposite approaches

### ✅ Solution: Cohesive Vision
```
Lo-fi bedroom pop, tape saturation, intimate vocals, organic drums,
vintage synth pads, DIY charm, authentic and unpolished
```
**Why it works:** All elements support the same aesthetic

---

### ❌ Problem: Too Long (1200+ characters)
```
[Insert massive paragraph with 50+ descriptors and contradictory ideas]
```
**Why it fails:** Suno truncates after ~950 characters, loses important details

### ✅ Solution: Focused & Concise
```
Hard-hitting trap beat, menacing 808s, rapid hi-hat rolls,
dark atmospheric pads, confident rap flow, modern production,
BPM: 75, Key: F Minor
```
**Why it works:** Under 950 chars, focused on core elements

## 🔥 Advanced Techniques

### Progressive Prompts (Arrow Technique)
Shows Suno the song's dynamic evolution:
```
Soft piano intro → gentle drums at 0:30 → 
builds with strings → explosive guitar-driven chorus at 1:00 →
stripped-down acoustic bridge → massive final chorus with full orchestration
```

### Layering Technique
Specify production layers for depth:
```
[Foundation] Punchy kick, deep 808 bass
[Rhythm] Syncopated hi-hats, snappy snare
[Melody] Lush pad chords, arpeggiated synth lead
[Vocals] Layered harmonies, ad-libs, subtle autotune
```

### Reference + Modification
Use artist references + unique twist:
```
Billie Eilish whisper vocals + The Weeknd dark production +
unexpected saxophone solo + trip-hop beats
```

### Genre Fusion Formula
Combine 2-3 genres with clear ratio:
```
70% Country storytelling + 20% Hip-Hop 808s + 10% Rock guitar riffs =
Modern country-rap fusion with edge
```

### Vocal Effects Specification
```
[Verse] Clean dry vocals, intimate delivery
[Chorus] Heavy reverb, doubled vocals, layered harmonies
[Bridge] Distorted megaphone effect, raw emotion
```

## 🎨 Use Case Examples

### For TikTok/Social Media Viral Potential
- High energy (8+/10)
- Catchy hook in first 8 seconds
- BPM: 120-140 (fits platform tempo)
- Modern production (2020s sound)

**Recommended:** [Pop Upbeat](prompts/pop.md#upbeat-dance-pop), [EDM Festival](prompts/edm.md#festival-banger)

---

### For Emotional Storytelling
- Lower energy (3-6/10)
- Piano or acoustic-driven
- BPM: 60-90 (reflective tempo)
- Space for lyrics to breathe

**Recommended:** [Pop Ballad](prompts/pop.md#emotional-ballad), [Country Narrative](prompts/country.md#storytelling-ballad)

---

### For Workout/Gym Playlists
- Very high energy (9-10/10)
- Driving rhythm
- BPM: 130-150 (workout-friendly)
- Aggressive or anthemic

**Recommended:** [Rock Anthem](prompts/rock.md#stadium-anthem), [Trap Banger](prompts/hip-hop.md#modern-trap)

---

### For Background/Study Music
- Low energy (1-3/10)
- No jarring elements
- BPM: 80-100 (calm tempo)
- Instrumental-focused

**Recommended:** [Jazz Lounge](prompts/jazz-blues.md#smooth-jazz), [Indie Dreamy](prompts/indie.md#bedroom-pop)
